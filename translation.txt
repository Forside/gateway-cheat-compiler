SPECIAL KEYPAD CODE
===================
As for the Special KEYPAD cheat code, the keypad value can be any combination of the following:

0x1      A
0x2      B
0x4     Select
0x8     Start
0x10    Right
0x20    Left
0x40    Up
0x80    Down
0x100    R
0x200    L
0x400    X
0x800    Y
